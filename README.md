# Fusion

Fusion은 Android 기기에서 로컬 LLM을 실행하고, 모델 라이브러리·벤치마크·메모리·대화 관리 기능을 실험하기 위한 온디바이스 AI 앱입니다.

현재 프로젝트는 알파 단계이며, 로컬 모델 실행 안정성, 저메모리 기기 최적화, 모델별 설정 관리, 장기 메모리 기능을 중심으로 개발 중입니다.

> Project Nuke의 실험적 Android 클라이언트입니다.

---

## 주요 기능

### 로컬 AI 채팅

- Android 기기에서 로컬 LLM 실행
- Gemma 기반 로컬 채팅 지원
- 모델별 생성 설정 적용
- Reasoning, 웹 검색, MTP 가속 실험 옵션
- 응답 속도와 토큰 정보 표시
- 답변 다시 생성 및 스타일 재생성 지원
  - 답변 다시 생성
  - 더 짧게
  - 더 자세히
  - 표로 정리
  - 전문가 톤

## 모델 라이브러리

Fusion은 여러 모델 후보를 관리하기 위한 Model Zoo 구조를 제공합니다.

지원/관리 후보:

- Gemma
- Qwen
- Llama
- Phi
- DeepSeek
- Mistral
- Kimi
- Custom Model

### 모델 관리 기능

- 모델 검색
- 모델 필터
- 모델 정렬
- 모델 즐겨찾기
- 숨긴 모델 관리
- 최근 사용 모델
- 모델별 메모
- 모델별 벤치마크 요약
- 모델 상세 정보
- 모델 페이지 링크
- 공식 페이지 링크
- 외부 모델 파일 연결
- 모델 저장공간 관리

### 기기별 추천

Fusion은 현재 기기의 메모리와 모델 크기를 기준으로 모델 추천 정보를 표시합니다.

예시 기준:

- 8GB 기기에서는 소형 모델 우선 추천
- 5GB 이상 모델은 기본 추천에서 제외
- 무거운 모델 선택 전 경고 표시
- 현재 사용 가능 메모리가 낮을 경우 실행 전 정리 안내

## 메모리 시스템

Fusion은 대화에서 중요한 정보를 저장하고, 이후 답변 생성에 참고할 수 있는 메모리 기능을 실험 중입니다.

### 대화 요약 메모리

- 대화별 요약 저장
- 긴 대화에서도 핵심 맥락 유지
- 요약 직접 편집
- 요약 복사
- 요약 삭제
- 답변 생성 및 답변 재생성 시 요약 반영

### 메모리 후보 추출

- 현재 대화에서 장기적으로 참고할 만한 정보 추출
- 자동 저장하지 않고 사용자 확인 후 저장
- 후보 복사
- 저장된 메모리 관리

### 메모리 관리

- 저장된 메모리 확인
- 메모리 수정
- 메모리 삭제
- 메모리 사용/사용 안 함 전환
- 메모리 컨텍스트 미리보기
- 답변 생성에 포함될 메모리 확인

## 데이터 관리

Fusion은 로컬 데이터 관리 기능을 제공합니다.

- 설정 백업
- 설정 복원
- 설정 JSON 복사
- 클립보드에서 복원
- 채팅 Markdown 내보내기
- 내보낼 채팅 선택
- 아카이브된 채팅 관리

설정 백업에는 앱 설정과 모델 라이브러리 설정이 포함될 수 있습니다.  
채팅 기록, 첨부파일, 모델 파일, 저장된 메모리 내용은 기본적으로 설정 백업에 포함하지 않는 방향으로 설계되어 있습니다.

## 개발자 도구

Fusion은 알파 테스트와 모델 디버깅을 위해 개발자 로그 화면을 제공합니다.

확인 가능한 정보:

- 현재 앱 상태
- 선택된 모델
- 런타임 형식
- 가속기 설정
- MTP 상태
- 메모리 상태
- 기기 정보
- SoC/AP 정보
- 벤치마크 요약
- 최근 오류 정보

개발자 로그와 오류 보고서는 클립보드로 복사할 수 있습니다.

개인정보 보호를 위해 다음 정보는 개발자 로그에 포함하지 않습니다.

- 채팅 전문
- 전체 프롬프트
- 첨부파일 내용
- API 키
- 저장된 메모리 내용
